# v0.1.0 - First Public Checkpoint Skill For AI Coding Worktree Hygiene

AI Worktree Hygiene is a tiny checkpoint skill for Codex, Claude Code, Cursor, and
other coding agents.

It targets a common AI-assisted coding failure mode:

```text
Tests pass, but the branch is not reviewable.
```

## 30-Second Before / After

Before:

```text
 M package.json
 M package-lock.json
 M src/dashboard/App.tsx
 M src/core/store.ts
?? .research/flowcheck/
?? .research/worktree-tools/
?? pnpm-lock.yaml

ignored:
dist/
coverage/
node_modules/
```

Run:

```text
Use ai-worktree-hygiene to check whether this branch is safe to continue, commit, or hand off.
```

After:

```text
research-summary -> commit separately
research-cache   -> discard or keep ignored
dependency       -> review package-manager drift
product-code     -> review and split mixed files
generated-runtime -> clean or record as local residue
```

Result:

```text
Not safe to push yet.
Safe next action: commit docs/research first, discard raw research cache,
resolve package-manager drift, then review product-code + tests as one lane.
```
