# Related Projects And Boundaries

AI Worktree Hygiene is intentionally small. It does not replace workspace managers,
MCP servers, or full code review platforms.

## FlowCheck

FlowCheck describes itself as a git hygiene safety layer for AI-first development.
It is closer to an MCP monitoring product.

AI Worktree Hygiene is smaller: a copyable skill and checkpoint template.

## Vibe Kanban

Vibe Kanban is a full workspace and task manager for coding agents.

AI Worktree Hygiene does not manage tasks, terminals, or dev servers. It helps review
what a branch contains before continuing, committing, or handing off.

## Parallel Worktree Tools

Parallel worktree tools help run multiple agents in isolated checkouts.

AI Worktree Hygiene helps classify what those agents leave behind.
